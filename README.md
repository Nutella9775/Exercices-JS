# Exercices de Base JavaScript
 
Ce dépôt est dédié aux exercices fondamentaux de JavaScript, organisé de manière logique.

## Objectifs d'apprentissage

Chaque dossier correspond à un ensemble de défis visant à maîtriser un concept spécifique :

* **Variables & Types :** `let`, `const`, String, Number, Boolean, `null`, `undefined`.
* **Opérateurs :** Arithmétiques, de comparaison (`==`, `===`), logiques (`&&`, `||`, `!`).
* **Flux de Contrôle :** Conditions (`if/else`, `switch`) et ternaire.
* **Boucles (Loops) :** `for`, `while`, `do/while`, et l'itération simple.
* **Fonctions :** Déclarations, expressions, fonctions fléchées (arrow functions), paramètres, `return`.
* **Tableaux (Arrays) :** Déclaration, accès, ajout/suppression, itération de base.
* **Objets (Objects) :** Propriétés, accès (dot notation vs bracket notation), méthodes.
